응답  클래스는 서버가 클라이언트에세 반환하는 HTTP 응답의 종류를 정의합니다.
개발자는 반환되는 데이터의 형식을 제어하고 특정 HTTP 응답의 동작을 세밀하게 조정할 수 있습니다.

1. JSONResponse:  클라이언트에게 JSON형식의 데이터를 반환합니다. 이 클래스는 PYTHON의 디그셔너리나 Pydantic 모델을
                    JSON 문자열로 반환하여 응답바디에 담아 전송합니다.

2. HTMLResopnse:  클라이언트에게 HTML형식의 데이터를 반환합니다.  주로 웹페이지의 내용을 반환할 때 사용합니다.

3. PlainTextResponse: 클라이언트에게 단순 텍스트 형식의 응답을 반환합니다. (로깅, 간단한 메시지 전달에 적합)

4. RedirectResponse: 클라이언트를 지정된 다른 URL로 리디렉션하는 HTTP 응답을 생성합니다.(사용자를 다른 페이지로 유도할 떄 유용)
